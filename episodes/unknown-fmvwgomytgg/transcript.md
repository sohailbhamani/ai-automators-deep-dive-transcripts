# Unknown

What if external agents like Claw Desktop were able to automatically discover, understand, and execute your make.com scenarios without you having to individually tie them all together. This is where make's new MCP server fits in. And it's really quick to get set up. All you do is go to API MCP access, go to add new token, MCP token, press add. From there, you can copy out this link. Then just copy out this code, and then within Claw Desktop, go to file and settings. Then go to edit with notepad. And this is where you'll then paste in that previous MCP URL. From there, save that file, then restart Claude, and then any scenarios that you have marked as on demand and activated and where you have the scenario input set up so that Claude will know what to send to the scenario. Then you'll be good to go. Claude should then see these tools and then you can interact with those right away. Such as in my example here, I'm getting it to add to ClickUp list. I've got it to generate an image and then upload an article with that image to WordPress. The image here is uploaded. We have all correctly formatted content. All this was done with this really minimal MCP integration. So check out the link if you want to learn more cuz we have a full video where I go through this in much more detail.
